CSS Property Reference

Position Property

Property Description
position Allows you to control the layout of the page.
Value Name Description
static Normal Flow Each block-level element sits on top of the next one.
relative Relative Positioning Moves an element relative to its normal position.
absolute Absolute Positioning The element is taken out of normal flow and doesn't affect the position of other elements on the page.
fixed Fixed Positioning Positions the element in relation to the browser window, which means it always stays in the same place even if the page is scrolled.
sticky Sticky Positioning Positions element based on user's scroll position. Relative until a given offset position is met in the viewport - then it "sticks" in place.

Related Properties

Property Description
z-index Used to control which element sits on top. The higher the number, the closer that element is to the front.
float Puts an element as far to the left or right as possible. Anything else will flow around this element. You should set width too.
clear Allows you to say that no element should touch the left or right-hand sides of a box.
